Manual ROI / Zone Calibration

Manual calibration is now the required runtime path. Draw normalized processing ROI, tripwire, zones, and optional subzones here, then save the config used by offline/live/demo runs.

Polygon shapes need at least 3 points. For entry_line, click exactly 3 points: line point 1, line point 2, then the IN-side anchor point.

Click on the preview to add normalized points. The browser renders the overlay locally; the runtime will use the saved normalized config.
